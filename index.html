About This Project

Solar System Quiz Game is a modern, interactive web application designed to make learning about our Solar System fun and visually immersive.

Features:

  • Animated, interactive solar system and twinkling starfield background
  • Professionally styled, glassmorphic interface
  • Challenging quiz with instant feedback and a skip option for tricky questions
  • Responsive design for all screen sizes
  • About and Contact modals with rich content and social links
Why this project?
Space exploration inspires curiosity and wonder! This quiz helps users of all ages to discover interesting facts about the Solar System in a playful, engaging way.

Technologies used: HTML, CSS (with glassmorphism & gradients), JavaScript, and Font Awesome.
